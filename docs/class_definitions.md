# Definición de clases

Este proyecto detecta Equipos de Protección Personal (PPE) en obras AECO.

Clases utilizadas:

helmet – casco de seguridad del trabajador  
vest – chaleco reflectante  
person – trabajador presente en la obra

Reglas de etiquetado:

- Se etiquetan personas visibles.
- Se etiquetan cascos parcialmente visibles.
- No se etiquetan reflejos ni posters.
